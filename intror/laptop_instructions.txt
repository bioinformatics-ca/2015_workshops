=========================
Laptop Setup Instructions
=========================

Programs to Install:

1) Install the latest version of R which can be downloaded from http://www.r-project.org/ If you need help on installing R see bullet 6) below for assistance.

2) Download and install the most recent version of R Studio desktop from http://www.rstudio.com/

3) Create a project directory for data files and source code for this workshop.

4) A robust internet browser such as Firefox or Safari or Chrome (Internet Explorer is not recommended).

5) A PDF viewer (Adobe Acrobat, Preview or equivalent) or make sure that you can read PDF files in your Web browser.

6) For help installing R, you may find the following directions helpful resources: Tutorial on how to install R on windows only look at the "getting started" section: http://www.biostat.wisc.edu/~kbroman/Rintro/Rwin.html
and MACS only look at the "getting started" section: http://www.biostat.wisc.edu/~kbroman/Rintro/Rmac.html
